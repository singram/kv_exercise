# KV data store exercise

Quick solution to the classic key-value interview question.


To start the repl;

    $ ruby ./kv.rb


To run in server mode;

    $ ruby ./kv.rb --mode server


To run in client mode;

    $ ruby ./kv.rb --mode client


To see all options;

    $ ruby ./kv.rb --help
